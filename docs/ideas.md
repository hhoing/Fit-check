# Ideas

아직 바로 만들지는 않지만, 나중에 검토할 아이디어를 모아둡니다.

## 입력

- 공고 캡처 이미지 업로드
- 여러 이미지를 한 번에 업로드
- PDF 공고 업로드
- 브라우저 확장 프로그램으로 현재 페이지 저장
- 모바일에서 공유하기로 공고 저장

## 관리

- 마감 임박 공고 자동 상단 고정
- 지원 가능성이 높은 공고 추천
- 회사별 그룹핑
- 직무별 그룹핑
- 스펙 키워드별 필터
- 관심도 별점
- 지원 준비 체크리스트

## 분석

- 내 이력서에서 부족한 키워드 추천
- 자소서에 넣을 수 있는 관련 경험 추천
- 공고별 예상 난이도
- 같은 회사의 다른 공고 비교
- 비슷한 공고끼리 묶기

## 알림

- D-7, D-3, D-1 알림
- 오늘 마감 공고 강조
- 캘린더 일정 추가
- 이메일 알림

## 외부 연동

- 잡플래닛 기업 후기 링크
- 원티드, 사람인, 잡코리아 공고 링크 보존
- 네이버 지도 또는 카카오맵 통근 시간
- Google Calendar 연동
